# アフィリエイトブログ記事作成ガイドライン

このドキュメントは、すべての自動生成記事が従うべきルールを定義しています。
会話がクリアされた後も、このファイルを読み込んで一貫性のある記事を生成してください。

## 1. 記事の基本要件

### 文字数
- **最小：3000字以上（絶対）**
- **推奨：3500～4500字**
- 句点「。」の数：最低100～130個必須

### 見出し構成
- H1（# タイトル）：1個
- H2（## 見出し）：7個以上必須
- 見出し下には最低10～15文を配置

### 表（Markdown形式）
- **最低2～3個必須**
- 複数の表を自然に配置
- 比較表、一覧表、手順表など、多様なタイプ

### キーワード埋め込み
- メインキーワード：本文に**8回以上**必須
- 関連キーワード：**10回以上**必須
- 合計150回以上の出現が目安

## 2. 記事構成（必須レイアウト）

```
# タイトル

[導入：読者の悩みを1～2文で、50～100字]

## 見出し1（基本知識。600～700字）
[内容]
[画像埋め込み]

## 見出し2（重要なポイント・メリット。700～900字）
[内容]
[表を含める]
[画像埋め込み]

## 見出し3（比較・選び方。複数の表推奨。700～900字）
[複数表を配置]
[画像埋め込み]

## 見出し4（詳細な使い方・実践例。600～800字）
[内容]
[画像埋め込み]

## 見出し5（さらに詳しい情報。400～500字）
[内容]

## 見出し6（よくある質問・FAQ。300～400字）
Q1.
A.
Q2.
A.

## 見出し7（まとめ・CTA。300～400字）
[まとめ＋行動喚起]

---

## 目次（記事長が3500字以上の場合は必須）
【目次】
1. [見出し1](#h2-1)
2. [見出し2](#h2-2)
...

---

## 関連記事
[関連記事セクション]

---

## 関連キーワード
[関連キーワード列表]
```

## 3. AIっぽさを完全に排除するルール

### 絶対禁止フレーズ
- 「いかがでしたか」
- 「ぜひご覧ください」
- 「～と思います」「～といえます」
- 「重要です」「大切です」
- 「このように」「その結果」
- 「注目されています」「話題になっています」
- 「こちら」「ご紹介します」
- 「できます」「できるようになります」
- 「考えられます」「言えるでしょう」「わかります」
- 「もう一度」「さらに詳しく」
- 「～より」（比較時：「～の方が」を使う）

### 文体のルール
- **「です」「ます」で統一**（「ある」「いる」と混ぜない）
- 1文は15～25字が最適
- 短すぎず長すぎず（最大30字を超えるなら段落を分ける）
- 同じ単語の繰り返しは避け、同義語で言い換える
- 「～のです」は避ける（～ですを使う）

### 文体の改善例

❌ AIっぽい：
> 金魚の飼い方には多くの選択肢があります。最初に知るべきポイントとしては、適切な水槽選びが重要です。できるだけ大きい水槽を選ぶことで、金魚がより健康的に成長することができるようになります。

✅ 改善版：
> 金魚を飼う際は、最初に水槽を選びます。45cm以上の水槽を用意することで、金魚は十分な活動スペースを得られます。水槽が大きいほど、水質の管理も安定しやすくなります。

## 4. 目次機能（3500字以上の記事）

### 実装方法
記事本文の最初に以下の形式で目次を配置：

```markdown
## 目次【目次を表示】

1. [見出し1テキスト](#h2-1)
2. [見出し2テキスト](#h2-2)
3. [見出し3テキスト](#h2-3)
...
```

### 要件
- クリック可能なリンク形式（Markdown リンク）
- IDは `#h2-1`, `#h2-2` などのパターン
- 表示/非表示のトグル機能は別途実装

## 5. ページ最後のセクション（必須）

### 関連記事セクション
```markdown
## 関連記事

このトピックに関連した他の記事：
- [記事タイトル1](リンク)
- [記事タイトル2](リンク)
- [記事タイトル3](リンク)
```

### 人気記事セクション
```markdown
## サイト内の人気記事

ブログ内でよく読まれている記事：
- [人気記事1](リンク)
- [人気記事2](リンク)
- [人気記事3](リンク)
```

### 関連キーワード
```markdown
## 関連キーワード

このテーマで検索されることが多いキーワード：
- キーワード1
- キーワード2
- キーワード3
```

## 6. 出力禁止内容

以下のコンテンツは記事に含めてはいけません：

- ✓ 高品質（テスト用マーカー）
- 完成
- ---（区切り線以降のメタデータ）
- **字数確認：** などの品質チェック情報
- 【関連記事】【人気記事】などの括弧つきマーカー
- 外部出力用のメタデータ
- HTMLコメント

## 7. 画像埋め込みルール

- 各見出し下に1枚以上の画像を配置
- 形式：`![説明文](/images/blog/2026-04-XX-xxxxx.jpg)`
- すべての画像を自動生成（別途スクリプトで実施）

## 8. 禁止ワードの完全リスト

以下のいずれかが1個でも含まれたら、その部分を修正して再度提示：

```
いかがでしたか、ぜひご覧ください、と思います、といえます、
重要です、大切です、このように、その結果、注目されています、
話題になっています、こちら、ご紹介します、できます、できるようになります、
考えられます、言えるでしょう、わかります、もう一度、さらに詳しく、
完成、✓、高品質、---、以下、まとめ（単独終了時）、のです、より（比較時）
```

## 9. 品質チェック（内部用：出力しない）

生成前に以下を確認：
- [ ] 文字数が3000字以上か
- [ ] 表が2個以上含まれているか
- [ ] 見出しが7個以上あるか
- [ ] メインキーワードが8回以上含まれているか
- [ ] 句点「。」が100個以上あるか
- [ ] 禁止ワードが0個か
- [ ] 末尾に不要なメタデータがないか

**チェック結果を記事に出力してはいけません。内部判定のみ。**
