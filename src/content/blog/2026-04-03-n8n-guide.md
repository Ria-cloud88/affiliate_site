「Zapierは高い」「Makeも使いこなせない」と感じていませんか？

結論：**n8nはオープンソースの自動化ツールで、自分のサーバーにインストールすれば完全無料・無制限で使えます**。

## n8nとは？

ノーコードで複数サービスを連携して自動化できるワークフローツールです。ZapierやMakeと同じジャンルですが、**オープンソース**なのが最大の特徴。

GitHubで公開されており、自分のサーバー（またはPC）で動かせば料金は一切かかりません。

対応サービス数はZapierに劣りますが、**カスタムコード（JavaScript/Python）が書けるノード**があるので、APIがあれば何でも連携できます。

## インストール方法

### Docker（推奨）

```bash
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  n8nio/n8n
```

ブラウザで `http://localhost:5678` を開くと使えます。

### npm

```bash
npm install n8n -g
n8n start
```

### Windows/Mac（デスクトップアプリ）

n8n公式サイトからデスクトップ版をダウンロードできます。インストールするだけで起動します。

## 基本操作

### ワークフローの作成

1. 左メニューの「Workflows」→「New Workflow」
2. 「+」ボタンでノードを追加
3. ノード同士を線でつなぐ
4. 各ノードの設定を入力
5. 「Execute Workflow」で実行

### トリガーの種類

- **Webhook**：外部からのHTTPリクエストで起動
- **Cron**：定期実行（毎日9時など）
- **サービストリガー**：Gmail受信、Slackメッセージなど

## おすすめ自動化レシピ

### 1. RSS → Notion自動保存

技術ブログの新着記事をNotionデータベースに自動保存します。

`RSS Feed Read` → `Notion（Create Page）`

### 2. Gmail → Slack通知

重要なメールが届いたらSlackに通知します。

`Gmail Trigger` → `IF（条件分岐）` → `Slack`

### 3. Google Sheets → 定期レポート

スプレッドシートのデータを毎週まとめてメール送信。

`Cron` → `Google Sheets` → `Gmail`

### 4. Webhook → Discord通知

GitHubのPRが作成されたらDiscordに通知。

`Webhook` → `Discord`

### 5. フォーム → CRM登録

Googleフォームの回答を自動でHubSpotに登録。

`Google Sheets Trigger` → `HubSpot`

## 便利なノード

### Codeノード

JavaScriptでカスタム処理を書けます。

```javascript
// 前のノードのデータを加工する例
const items = $input.all();
return items.map(item => ({
  json: {
    name: item.json.name.toUpperCase(),
    timestamp: new Date().toISOString()
  }
}));
```

### HTTP Requestノード

APIエンドポイントに直接リクエストを送れます。n8nに対応していないサービスでも、APIがあれば連携できます。

### IF・Switchノード

条件によって処理を分岐させます。「金額が1万円以上なら承認フロー、未満なら自動処理」といった使い方ができます。
